Privacy Policy

Last updated: February 9, 2026

Overview

Context Marker is a Chrome extension that visually marks browser tabs based on user-defined rules to help distinguish between environments (e.g., production, staging, development).

Data Collection

Context Marker does not collect, transmit, or share any personal data. All processing happens locally in your browser.

Data Storage

The extension stores the following data locally using Chrome's storage.local API:

  • Rules: User-defined rules for environment detection (host patterns, extractors, display settings).
  • Display style preference: The selected overlay display style (top bar, ribbon, or combo).

This data never leaves your browser and is not sent to any external server.

Page Content Access

The extension reads page content (DOM text, meta tags, page title) solely to evaluate user-defined detection rules. This information is processed in memory, is not stored persistently, and is not transmitted anywhere.

Permissions

  • <all_urls>: Required to run detection rules on any website the user configures. No data is collected from these pages.
  • storage: Used to persist user rules and preferences locally.

Third-Party Services

Context Marker does not use any third-party services, analytics, or tracking.

Changes to This Policy

Any changes to this privacy policy will be reflected on this page with an updated date.
